Expand description

Represents a private chat.

Fields§

§id: ChatPeerId

Unique identifier of the chat.

§first_name: String

First name of the other party.

§accent_color: Option<AccentColor>

Identifier of the accent color for the chat name and backgrounds of the chat photo, reply header, and link preview.

Returned only in crate::types::GetChat.

§active_usernames: Option<Vec<String>>

List of all active usernames of the other party.

Returned only in crate::types::GetChat.

§background_custom_emoji_id: Option<String>

Custom emoji identifier of emoji chosen by the chat for the reply header and link preview background.

Returned only in crate::types::GetChat.

§bio: Option<String>

Bio of the other party.

Returned only in crate::types::GetChat.

§emoji_status_custom_emoji_id: Option<String>

Custom emoji identifier of emoji status of the other party.

Returned only in crate::types::GetChat.

§emoji_status_expiration_date: Option<i64>

Expiration date of the emoji status of the other party in a private chat in Unix time, if any.

Returned only in crate::types::GetChat.

§has_private_forwards: Option<bool>

Indicates whether privacy settings of the other party in the private chat allows to use tg://user?id=<user_id> links only in chats with the user.

Returned only in crate::types::GetChat.

§has_restricted_voice_and_video_messages: Option<bool>

Indicates whether privacy settings of the other party restrict sending voice and video note messages.

Returned only in crate::types::GetChat.

§last_name: Option<String>

Last name of the other party.

§message_auto_delete_time: Option<i64>

The time after which all messages sent to the chat will be automatically deleted; in seconds.

Returned only in crate::types::GetChat.

§photo: Option<ChatPhoto>

Photo of the other party.

Returned only in crate::types::GetChat.

§pinned_message: Option<Box<Message>>

Latest pinned message.

Returned only in crate::types::GetChat.

§profile_accent_color: Option<ProfileAccentColor>

Identifier of the accent color for the chat’s profile background.

Returned only in crate::types::GetChat.

§profile_background_custom_emoji_id: Option<String>

Custom emoji identifier of the emoji chosen by the chat for its profile background.

Returned only in crate::types::GetChat.

§username: Option<ChatUsername>

Username of the target chat.
