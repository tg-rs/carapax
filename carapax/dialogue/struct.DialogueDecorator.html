Struct DialogueDecorator

pub struct DialogueDecorator<B, H, HI, HS> { /* private fields */ }
Available on crate feature dialogue only.
Expand description

A decorator for dialogue handlers.

An inner handler must return a DialogueResult. The decorator will automatically load and save the state of the dialogue returned in the result.

Note that you need to register a crate::session::SessionManager instance in the crate::Context. Otherwise the decorator will return an error indicating that the session manager is not registered.
